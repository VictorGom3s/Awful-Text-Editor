# Awful-Text-Editor

### Simple CLI text editor in C with functions to type, print on screen, insert text and delete text.

## How to run (Unix)

- Compile the source with:

  `$ gcc ate.c`

- Execute the program and provide some text:

  `$ ./a.out Some Random Text`
